# Starcraft-2-IA-Pysc2
Starcraft 2 IA Zerg Bot using PySC2 written in Python


## Description

The AI use an aggressive strategy to try to win the game.

## Getting Started

### Dependencies

* Pysc2
* Starcraft II


### Executing program

* Run SC2IA.py
```
python SC2IA.py
```
